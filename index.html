About Om's Docs

Om's Docs is a personal document management system built for simplicity and efficiency. Store, browse, and view your files right in the browser — no complicated setup required.

Features

  •  Store documents locally in your browser
  •  View PDFs with page-by-page navigation
  •  Preview images and plain-text files
  •  Print and download any stored document
  •  Search and filter your document library
  •  Drag & drop file adding
  •  Dashboard with file statistics
