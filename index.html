About Om's Docs

Om's Docs is a personal document management system. Files are securely stored in Appwrite Cloud (Singapore region) and accessible from any device.

Features

  •  Store documents in Appwrite Cloud
  •  View PDFs with page-by-page navigation
  •  Preview images and plain-text files
  •  Print and download any stored document
  •  Search and filter your document library
  •  Drag & drop file adding
  •  Dashboard with file statistics
  •  Cross-device access via cloud storage

Storage

Files are stored in Appwrite Cloud Storage (Bucket: files.om.docs) and metadata is persisted in localStorage as an index cache.
